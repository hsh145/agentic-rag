Agentic RAG 系统测试文档
==========================

概述
----
本文档用于测试 Agentic RAG 系统的文本解析能力。

系统功能
--------
1. 多格式文件解析（PDF、Word、Excel、图片、文本）
2. 语义分块与向量索引
3. 混合检索（向量 + BM25 + RRF）
4. LangGraph Agent 工作流
5. FastAPI 服务

技术栈
------
- 语言：Python 3.11+
- 框架：LangGraph + FastAPI
- 向量检索：FAISS + DashScope Embedding
- 关键词检索：BM25

评估指标
--------
以下是系统需要衡量的关键指标：

1. 文档解析成功率：成功解析的文件比例
2. 文本提取率：提取文本与预期文本的比例
3. 检索召回率（Recall）：相关文档被检索到的比例
4. 忠实度（Faithfulness）：回答是否忠于上下文

测试数据
--------
这些指标需要通过标准化的测试数据集来评估。
